# How do I manage security deposits?

Source: https://agodahomeshelp.zendesk.com/hc/en-us/articles/360023687854-How-do-I-manage-security-deposits

Updated: 2023-07-17T09:29:58Z

Renting out your own place is a personal experience -- so what happens if your property or belongings are damaged by guests?

Agoda lets you set a security deposit policy that allows you to collect upfront money from guests to cover potential property damage. You have full control over the payment process, deciding when and how to collect the deposit from guests. Please note that Agoda does not handle the security deposit directly.
