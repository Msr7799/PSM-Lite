# Why can I not see my property type when creating a property?

Source: https://agodahomeshelp.zendesk.com/hc/en-us/articles/360023968673-Why-can-I-not-see-my-property-type-when-creating-a-property

Updated: 2022-12-15T10:13:49Z

Currently the types of homes we support for guests and hosts are entire apartment, entire bungalow, entire villa, and entire house. If you have a different kind of accommodation type, you can list it through your Hotel platform, YCS.

Our Hotel hosting site is **ycs.agoda.com **
