# Agoda Inbox Upgrade: Faster, Smarter, More Convenient Messaging for Hosts

Source: https://agodahomeshelp.zendesk.com/hc/en-us/articles/42562650843027-Agoda-Inbox-Upgrade-Faster-Smarter-More-Convenient-Messaging-for-Hosts

Updated: 2025-06-29T12:43:53Z

We’ve been working hard to improve your Agoda Inbox experience, making it more powerful, efficient, and easy to use. Here's an overview of the exciting new features now available, along with upcoming enhancements designed to help you save time and communicate with guests more effectively.


**What’s New in Your Agoda Inbox**

Improved Mobile Experience

The Agoda Inbox on the mobile app has been significantly upgraded, providing a faster, smoother messaging experience — perfect for managing communication on the go.


**New Features Available Now:**

-
**Image Attachments**: Easily send images to guests directly through the inbox.
-
**Quick Replies with Templates**:
- Under the message text box, you’ll now see the option to quickly insert a saved reply.
- Simply click to view your existing templates or create a new one.
- Selecting a template will automatically place the message in the text box — saving you time, just like the existing template feature on desktop.

-
**Instant Message Translations**:- When a guest sends a message in a different language from your selected language, you’ll now see the option to instantly translate the message for ease of communication.


**Enhanced Design and Stability**

- A cleaner, more intuitive interface for smoother navigation.
- Bug fixes to ensure reliable, consistent performance.


**Coming Soon**

We're in the process of rolling out additional time-saving features:

- Scheduled Messages: Prepare messages in advance and send them automatically at your preferred time.
- Auto-Replies: Respond to guests even when you’re unavailable — effortlessly.


**Why These Changes Matter**

We know your time is valuable. These upgrades are designed to help you:

- Communicate with guests more efficiently
- Reduce repetitive typing
- Overcome language barriers
- Stay responsive anytime, anywhere
