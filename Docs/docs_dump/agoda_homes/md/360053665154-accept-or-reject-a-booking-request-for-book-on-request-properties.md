# Accept or reject a booking request for Book-on-Request properties

Source: https://agodahomeshelp.zendesk.com/hc/en-us/articles/360053665154-Accept-or-reject-a-booking-request-for-Book-on-Request-properties

Updated: 2024-08-28T13:39:53Z

If your propertyâ€™s reservation type** **is set up as **Require reservation requests**, every booking request will be sent to you via email for you to accept/decline within 24 hours, else the booking will be automatically cancelled. Please note that Book-on-Request reservations create a suboptimal experience for guests as they need to wait for a day to receive a booking confirmation. We encourage you to keep your property reservation type as **Instant book**. If you need to maintain an on-request reservation type, please monitor these reservation request notifications so that you can promptly respond to guests.

Once you accept** **a booking, you will receive an email notification from us stating that the booking is confirmed. If you decline the booking request, the booking will be cancelled. If you have missed the booking request email notification, please go to **Messages** on Host Manage to accept/decline the booking.

All booking statuses are recorded on Host Manage.

To check a reservation request, go to the **Reservations**** **page and click on the **Request **tab where you can either **Decline **or **Accept **a booking request. If you are unable to accept/decline the booking within 24 hours, the booking will be automatically declined.
