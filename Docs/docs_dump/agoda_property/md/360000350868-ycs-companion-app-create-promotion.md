# YCS Companion App -- Create Promotion

Source: https://agodapropertypartnerhelp.zendesk.com/hc/en-us/articles/360000350868-YCS-Companion-App-Create-Promotion

Updated: 2021-07-30T01:52:31Z

Express Tonight screen allows you to manage Express Tonight promotion which will help you to offload soon-to-expire last rooms quickly and easily.

You can create your new Express Tonight promotion using the pre-filled promotion template and customize various parameter to fit your needs. Once done, you can see the details of the promotion you have created.

If you have any existing Express Tonight promotion for your property, the app will display the list of your promotion in Express Tonight screen. You can click on the promotion to see the details, update, or delete the promotion.
