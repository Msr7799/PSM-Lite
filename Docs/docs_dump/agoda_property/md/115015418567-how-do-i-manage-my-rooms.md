**How to add a room?**

Here are the steps how to add new room type. Please provide as many details as possible.

- Log in to YCS
- Go to
**Setting** and select **Room Setup**
- Click
**Add Room Button**
**Fill in details**
- Click
** Save**





**Explanation of each field**

**Basic Info**

-
**Room Name:** Enter the first letter of the room name. You must select from the dropdown list
-
**External Code ID:** Enter the ID provided by your property or channel manager to help you map rooms
-
**No. of Rooms:** No of the rooms for this room type that are available
-
**Minimum Rate:** Enter the lowest possible rate for this room. This is to ensure you do not enter a rate lower than the minimum when uploading the rates.

**Occupancy Settings**

-
**Persons:** Select the number of persons that can stay in the room based on the existing beds.
-
**Extra Beds:** Select the number of extra beds that are allowed in this room. (Maximum one extra bed is supported)
-
**Baby Cots:** Select the number of baby cots/cribs that are allowed in this room.
-
**Allow children in the room type:** Provide information if children can stay in this room type. For child age policy set up click here.

**Content**

-
**Upload photos:** You must upload at least 1 photo for each room with a minimum resolution of 800 x 600 pixels and a maximum file size of 4 MB.
-
**Room Size:** Enter the average size of this room type in sq.m.
-
**View:** Select the view of this room.
-
**Amenities:** Select all amenities that are available in this room.
-
**Bedding:** Select the number and types of beds found in this room.
-
**Bathrooms:** Select the number of bathrooms available in this room


**How to activate/deactivate room?**

- Go to
** Settings** and select **Room Setup**
- Click the
** greyed box** to activate the room/ Click the **green box** for deactivation.
- Click
** OK** and **Save**.


**How to add/update room name?**

If you would like to change the name of existing room, below is the steps;

- Go to
**Settings** and select **Room Setup**
-
**Click on the room** that you would like to amend the name
-
**Under Room Name field** - remove existing room name and start typing (Start from the most important keyword, then add other terms to get the room name you need. Be patient! The list can be quite long)
-
**Select a standard room name **from the drop-down list
- Click
**Save**

Check out more tips to **find a suitable room name** and why we recommend using **standard room names**.


- For existing bookings, updating 'Room name' will not change the actual name in the booking voucher sent to customers
-
Updating the 'Room name'
** **will likely affect upcoming reservations. Remember to check your upcoming bookings first.
- If your property is using a channel manager, please contact your channel manager partner to remove/add the mapping on their side.
