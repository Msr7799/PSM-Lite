In order to be bookable on our website, you need to set up all your room details, including basic info, occupancy, content, and offers.

Please set up **at least one room** to activate your property. Be sure you provide as many details as possible.



**Basic Information**

|
|

|

Select the number of baby cots/cribs allowed in this room. |
