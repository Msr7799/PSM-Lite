**How to Update My Allotment?**

- Go to
**Rate & Allotment** and select **View**
- Select
** Room Type **and **Rate Plan **that you want to update
- Select the
** date range** that you would like to adjust
- On the right window, change the
**number of available rooms** or swipe the toggle to **grey color if you want to block** or **green color to make it available**
- Click
** Save**




**What is Closed to Arrival?**

In case you want to restrict the guests not to arrive / check-in on specific date range, you can set this value on the calendar page. If guest's stay includes this date but does not begin on it, then your room would be bookable on the website. To set close to arrival, here's how:

- Go to
**Rate & Allotment** and select **View**
- Select
**Room Type and Rate Plan **that you want to apply
- On the right window, select
**date range**
- Go to
**Advanced Tab** and tick on **Close to arrival**
- Click
**Save**


#
**What is Closed to Departure?**

In case you want to restrict the guests not to depart / check-out on specific date range, you can set this value on the calendar page. If guest stay includes this date but does not end on it, then your room would be bookable on the website. To set close to departure, here's how:

- Go to
**Rate & Allotment** and select **View**
- Select
**Room Type and Rate Plan **that you want to apply
- On the right window, select
**date range**
- Go to
**Advanced Tab** and tick on Close to departure
- Click
**Save**


**Frequent Asked Questions**

**What if I use a channel manager?**

If you work with a channel manager, you have to update your rates allotments in their system. Please contact their support team in case you need any assistance.


**Do I need to manage inventory all the time?**

As per our agreement, it is your responsibility to keep our inventory up to date at all times.
